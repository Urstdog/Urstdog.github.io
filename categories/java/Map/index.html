1.Map 接口介绍

  1. Map 与 Collection 并列,保存的是映射关系的Key-Value,Map 无序存储
  2. Map 中的 key 和 value 可以是任何引用类型的数据,会分装到HashMap$Node对象中
  3. Map 中的 key 不允许有重复值,value 可以重复,重复加入相同的 key 会进行替换 value
  4. key 可以为 null,只能有一个,而 value 可以有多个
  5. key 和 value 之间存在单向一对一关系,通过指定的 key 可以找到 value
  6. 为了方便遍历,Map 会创建 EntrySet 集合 ,该集合存放的元素是 Entry, 一个 Entry 对象 存放 key-value

2.Map 常用方法

  1. put(key,value) //添加 k-v 对
  2. size() //返回元素数量
  3. get(key) //由 key 值得到对应 value 值
  4. remove(key) //删除 k-v 对
  5. isEmpty() //判断是否为空
  6. containsKey(key) //判断集合中是否含有指定 key
  7. containsValue(value) //判断集合中是否含有指定 value
  8. getOrDefault(key,0) //获取指定 key 对应的 value,如果不存在用设置的默认值代替

3.Map 中的 enrty

  1. entry里面存放的是 key 和 value,但真正的 key 和 value 存放在 HashMap$Node对象中,entry只是引用了HashMap$Node 中的 key 和 value
  2. entrySet 里面存放的是 entry,是 Set 集合

4.Map 集合的遍历方式

1. 通过 KeySet()方法,再用增强 for 循环遍历

2. 通过 keySet()方法,再用迭代器遍历

3. 通过 entrySet()方法,再用增强 for 循环遍历

4. 通过 entrySet()方法,再用迭代器遍历
